# Social Media Posts
## UNIVERSELLE DESIGN-REGELN — GILT FÜR JEDEN POST                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                                                                       
  ### Grundprinzip                                                                                                                                                                                                                                                                                     
  Jeder Post – egal ob Carousel, Single Image, Story oder Reel-Thumbnail –                                                                                                                                                                                                                             
  folgt denselben Regeln. Kein Element darf "schweben", kein Abstand darf                                                                                                                                                                                                                              
  zufällig sein, kein Slide darf anders aussehen als der Rest.                                                                                                                                                                                                                                         
                                                                                                                                                                                                                                                                                                       
  ### Festes Grid-System                                                                                                                                                                                                                                                                               
                                                                                                                                                                                                                                                                                                       
  **Format 1: Hochformat (1080x1350px) — Carousel + Feed**                                                                                                                                                                                                                                             
  **Format 2: Quadrat (1080x1080px) — Single Image**                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                                                                       
  | Regel | Wert | Gilt für |                                                                                                                                                                                                                                                                          
  |-------|------|----------|                                                                                                                                                                                                                                                                          
  | Padding links | 72px | Alles |                                                                                                                                                                                                                                                                     
  | Padding rechts | 72px | Alles |                                                                                                                                                                                                                                                                    
  | Padding oben (Safe Zone) | 60px | Alles |                                                                                                                                                                                                                                                          
  | Padding unten (Safe Zone) | 40px | Alles |                                                                                                                                                                                                                                                         
  | Logo "socialeap." Position | Unten rechts, 72px vom Rand | Alles |                                                                                                                                                                                                                                 
  | Max. Textbreite | 936px (1080 - 2×72) | Alles |                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                                       
  ### Vertikale Verteilung — KEIN toter Raum                                                                                                                                                                                                                                                           
                                                                                                                                                                                                                                                                                                       
  Das häufigste Problem: Content klebt oben, unten ist alles leer.                                                                                                                                                                                                                                     
                                                                                                                                                                                                                                                                                                       
  **Regel:** Der Content muss den verfügbaren Raum zwischen Safe Zone oben                                                                                                                                                                                                                             
  und Logo unten GLEICHMÄSSIG nutzen. Kein Bereich darf mehr als 25% der                                                                                                                                                                                                                               
  Slide-Höhe komplett leer sein — es sei denn, es ist ein bewusster                                                                                                                                                                                                                                    
  Designentscheid auf dem Cover- oder CTA-Slide.                                                                                                                                                                                                                                                       
                                                                                                                                                                                                                                                                                                       
  **So verteilst du Content vertikal:**                                                                                                                                                                                                                                                                
  - Messe den verfügbaren Raum: Slide-Höhe minus Safe Zones minus Logo-Bereich                                                                                                                                                                                                                         
  - Verteile die Elemente gleichmäßig in diesem Raum                                                                                                                                                                                                                                                   
  - Wenn der Content nicht reicht: Elemente nach unten verschieben,                                                                                                                                                                                                                                    
    NICHT oben zusammenpressen                                                                                                                                                                                                                                                                         
                                                                                                                                                                                                                                                                                                       
  ### Abstände zwischen Elementen — FEST und KONSISTENT                                                                                                                                                                                                                                                
                                                                                                                                                                                                                                                                                                       
  | Übergang | Abstand |                                                                                                                                                                                                                                                                               
  |----------|---------|                                                                                                                                                                                                                                                                               
  | Nummer → Headline | 0px (direkt übereinander) |                                                                                                                                                                                                                                                    
  | Headline → Underline | 12px |                                                                                                                                                                                                                                                                      
  | Underline → Body-Text | 28px |                                                                                                                                                                                                                                                                     
  | Body-Text → Box/Card | 40px |                                                                                                                                                                                                                                                                      
  | Box/Card → nächstes Element | 40px |                                                                                                                                                                                                                                                               
  | Headline → Subline (wenn keine Underline) | 16px |                                                                                                                                                                                                                                                 
  | Subline → Body-Text | 24px |                                                                                                                                                                                                                                                                       
                                                                                                                                                                                                                                                                                                       
  Diese Abstände gelten für JEDEN Post-Typ. Niemals variieren.                                                                                                                                                                                                                                         
                                                                                                                                                                                                                                                                                                       
  ### Underlines und Akzentlinien                                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                                                                       
  - Dicke: IMMER 4px                                                                                                                                                                                                                                                                                   
  - Länge: So breit wie das längste Wort der Headline darüber                                                                                                                                                                                                                                          
  - Farbe: Akzentfarbe des Posts (Blau oder Orange, abwechselnd)                                                                                                                                                                                                                                       
  - Keine abgerundeten Enden — scharfe Kanten                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                       
  ### Boxen, Cards und hervorgehobene Bereiche                                                                                                                                                                                                                                                         
                                                                                                                                                                                                                                                                                                       
  - Border: 1.5px solid in Akzentfarbe                                                                                                                                                                                                                                                                 
  - Border-Radius: 12px                                                                                                                                                                                                                                                                                
  - Hintergrund: rgba(255,255,255,0.04)                                                                                                                                                                                                                                                                
  - Innerer Padding: 24px oben/unten, 28px links/rechts                                                                                                                                                                                                                                                
  - Breite: Volle Textbreite (936px)                                                                                                                                                                                                                                                                   
  - Position: IMMER an derselben Y-Position innerhalb einer Serie

  ### Farbwechsel-System bei Serien                                                                                                                                                                                                                                                                    
  - Ungerade Elemente (01, 03, 05): Akzent = Blau (#52C8F5)                                                                                                                                                                                                                                            
  - Gerade Elemente (02, 04): Akzent = Orange (#F5A028)                                                                                                                                                                                                                                                
  - Bei Single Images: Abwechselnd Blau und Orange von Post zu Post                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                                       
  ### Logo-Placement                                                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                                                                       
  - Position: IMMER unten rechts                                                                                                                                                                                                                                                                       
  - Abstand vom rechten Rand: 72px                                                                                                                                                                                                                                                                     
  - Abstand vom unteren Rand: 40px                                                                                                                                                                                                                                                                     
  - Format: "socialeap." in Sora Bold 24px, Punkt in Orange (#F5A028)                                                                                                                                                                                                                                  
  - NIEMALS woanders platzieren, NIEMALS weglassen                                                                                                                                                                                                                                                     
                                                                                                                                                                                                                                                                                                       
  ### VERBOTEN — auf keinem Post jemals                                                                                                                                                                                                                                                                
                                                                                                                                                                                                                                                                                                       
  - [ ] Mehr als 25% zusammenhängende Leerfläche ohne Designgrund                                                                                                                                                                                                                                      
  - [ ] Elemente die unterschiedliche Abstände zum Rand haben                                                                                                                                                                                                                                          
  - [ ] Texte die den 936px Textbereich sprengen oder nicht ausfüllen                                                                                                                                                                                                                                  
  - [ ] Fonts die nicht Sora oder Plus Jakarta Sans sind                                                                                                                                                                                                                                               
  - [ ] Farben die nicht in der Brand-Palette stehen                                                                                                                                                                                                                                                   
  - [ ] Boxen/Cards ohne festen inneren Padding                                                                                                                                                                                                                                                        
  - [ ] Logo vergessen oder an falscher Position                                                                                                                                                                                                                                                       
  - [ ] Unterschiedliche Abstände bei gleichen Elementtypen                                                                                                                                                                                                                                            
    innerhalb einer Serie (z.B. Headline-Abstand auf Slide 2                                                                                                                                                                                                                                           
    anders als auf Slide 4)                                                                                                                                                                                                                                                                            
  - [ ] Underlines mit unterschiedlicher Dicke                                                                                                                                                                                                                                                         
  - [ ] Texte kleiner als 20px (nicht lesbar auf Mobile)                                                                                                                                                                                                                                               
                                                                                                                                                                                                                                                                                                       
  ### QUALITÄTSPRÜFUNG — vor jedem Export                                                                                                                                                                                                                                                              
                                                                                                                                                                                                                                                                                                       
  1. Alle Slides/Posts nebeneinander legen (mental oder als Grid)                                                                                                                                                                                                                                      
  2. Prüfen: Sitzen gleiche Elemente auf EXAKT derselben Y-Position?                                                                                                                                                                                                                                   
  3. Prüfen: Ist der Content vertikal sauber verteilt — kein Loch unten?                                                                                                                                                                                                                               
  4. Prüfen: Sind alle Abstände identisch?                                                                                                                                                                                                                                                             
  5. Prüfen: Ist das Logo auf jedem Post an derselben Stelle?                                                                                                                                                                                                                                          
  6. Prüfen: Wäre ein Markendesigner mit dem Ergebnis zufrieden?                                                                                                                                                                                                                                       
                                                                                                                                                                                                                                                                                                       
  Wenn eine dieser Prüfungen fehlschlägt: Nochmal bauen.
